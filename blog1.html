HTML Tags which Everyone Ignores

- Pranay Dhanuka


September 2021

Every developer mostly use VS code to code their website. Most of them use the 'Emmet Abbreviation' plugin to lay down the basic structure of HTML file which is done via 'html:5'. This plugin made our life so simple. But due to this plugin we ignored to understand the meaning of tags used in it. Out of curiosity I searched about it and thought of writing a blog on it.

1.

!DOCTYPE

Every HTML document must start with a declaration.
It informs the browser of what document type to expect.

2.

html lang="en"

The lang attribute is used to set the language of text of the webpage.
Some common examples are "en" for English, "fr" for French and so on.

3.

meta charset="UTF-8"

The charset attribute informs the character encoding to the HTML document.
UTF stands for Unicode Transformation Format 8-bit. We use UTF-8 since it covers all the character and symbols in the world.

4.

meta name="viewport" content="width=device-width, initial-scale=1.0"

This tag is used to make page responsive. Initally viewport were not used, but after the introduction of mobile, tablets screen size isn't fix anymore, due to which viewport come in use, it makes the website more readable.
