CSS Box Model

- Pranay Dhanuka


September 2021

The CSS Box Model is used to create a definition for the way the HTML elements are organized on the screen. It help us to organize our data and make the website layout neat.
Each element can be thought of as having its own box. All the elemets of the box have to work together with each other, due to which it becomes importat to understand how BOX MODEL works.
The CSS Box Model contain 4 elements within it.

  • Margin
  • Border
  • Padding
  • content

    • Margin

    The margin area extends the border area to include empty space area to help seprating the element from the neighbor elements.
    The size of the margin area is determined by the
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

    • Border

    The border area extends the padding area to include elements borders.

    Padding

    The padding area helps us to include space between the content
    Padding is determined by the
  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

    • Content

    The Content area holds the main content of the element, such as text, images,video,etc.
    The content area's size can be defined with
  • width
  • min-width
  • max-width
  • height
  • min-height
  • max-height
